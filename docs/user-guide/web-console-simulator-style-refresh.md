# Web 控制台模拟器风格统一改造使用说明

## 1. 这次页面变化了什么

Web 控制台已经按当前设备模拟器的视觉风格统一收口，主要变化如下：

- 整体改为浅色、实体卡片式布局
- 左侧菜单不再是深色风格
- 顶部头部不再使用毛玻璃效果
- 各业务页面标题区统一为卡片式 `PageHeader`
- 登录页和仪表盘首页使用新的入口和概览布局

## 2. 登录页现在怎么用

- 左侧保留品牌信息和圆形轨道式联网动效
- 右侧只保留登录表单
- 仍然支持“账号密码登录”和“手机号登录”
- 登录成功后会自动跳转到当前账号可访问的首页

登录方式没有变化，但页面只保留登录必需信息和输入动作。

## 3. 主界面现在是什么风格

- 左侧菜单、顶部头部、标签页和内容区域都统一为浅色实体卡片风格
- 菜单选中态和重点操作统一使用浅蓝色强调
- 页面整体更接近桌面端工具，而不是营销式后台模板

## 4. 仪表盘现在怎么看

- 顶部先看平台概览和核心指标
- 中间看在线率、离线设备、产品数、告警和规则等实体指标卡
- 底部左侧看最近告警
- 底部右侧看告警级别分布和设备产品分布

这样可以先看主指标，再看明细，不需要在大量彩色卡片之间来回切换注意力。

## 5. 业务页面头部有什么变化

多数业务页面的标题区已经统一为新的卡片式头部：

- 标题更醒目
- 描述文案位置固定
- 右侧操作按钮会跟头部一起排布

这类变化不影响原有业务操作。

## 6. 使用建议

- Web 控制台现在更偏 PC 场景，建议在桌面浏览器中使用。
- 如果切换页面后仍看到旧样式，可刷新浏览器缓存后重新进入。
