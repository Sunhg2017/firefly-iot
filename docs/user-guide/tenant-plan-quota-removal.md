# 租户套餐与配额能力下线使用说明

> 适用角色: 系统运维、平台管理员
> 日期: 2026-04-06

## 1. 变更说明

当前版本已经移除租户套餐和租户配额能力。

你在“租户管理”页面中将不再看到：

- 套餐筛选
- 套餐统计卡片
- 套餐列
- 调整套餐
- 调整配额

## 2. 现在还能做什么

“租户管理”当前保留以下能力：

- 新建、编辑、启用、暂停、注销租户
- 设置租户隔离级别
- 查看租户实时用量和趋势
- 维护空间授权
- 维护 OpenAPI 订阅
- 维护租户 Webhook
- 重置租户超级管理员密码

## 3. 如何判断页面是否已更新

- 新建租户时，不再要求选择套餐。
- “更多”菜单中不再有“调整套餐”“调整配额”。
- 租户自助侧不再提供配额查看入口。

如果你仍看到旧入口，请刷新页面并重新登录。
