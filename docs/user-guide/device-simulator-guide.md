# 设备模拟器使用说明

## 1. 新建 Video 设备

1. 点击 `新建`
2. 选择 `Video`
3. 选择摄像头产品或填写 `ProductKey`
4. 选择 `GB28181 / RTSP / RTMP`
5. 补充对应参数

## 2. Video 模式怎么连接

点击 `连接` 后，模拟器会先同步平台设备资产，再执行媒体控制。

同步顺序：

1. 通过 `DEVICE` 路由查重
2. 创建或更新视频设备资产
3. 保存 `platformDeviceId`
4. 后续控制都使用这个 `platformDeviceId`

## 3. GB28181 怎么调

1. 填写 `GB 设备编号`
2. 填写 `GB 域`
3. 填写设备级 `SIP 密码`（必填）
4. 连接后执行 `SIP 注册`
5. 需要时再执行 `开启心跳`

## 4. RTSP / RTMP 怎么调

1. 选择 `RTSP` 或 `RTMP`
2. 填完整 `sourceUrl`
3. 点击连接
4. 连接成功后可执行播放、截图、录像

## 5. 常见问题

### 5.1 为什么第二次连接没有再创建新设备

因为模拟器会优先复用同一台设备的 `platformDeviceId`。

### 5.2 为什么通道列表现在不走 MEDIA

因为通道是设备资产主数据，已经改为走 `DEVICE` 路由。

### 5.3 为什么页面里不再显示“平台视频设备”

因为现在绑定的是统一设备资产，所以口径统一改成 `平台设备资产`。
