# 设备模拟器平台登录态物模型使用说明

## 1. 使用前提

- 已在设备模拟器中配置好环境
- 已切换到目标环境
- 已登录当前环境对应的租户账号

## 2. 现在怎么同步物模型

1. 在左侧选择一个 HTTP、MQTT、CoAP 或 Video 模拟设备。
2. 填写正确的 `ProductKey`。
3. 在右侧查看“平台物模型”卡片。
4. 系统会自动按当前环境登录态同步物模型。
5. 需要重新拉取时，点击“刷新”。

## 3. 不再需要什么

以下内容不再需要手工配置：

- OpenAPI 网关地址
- Access Key
- Secret Key

模拟器会直接使用当前环境配置的网关地址和当前登录态。

## 4. 同步失败怎么办

- 提示“请先登录当前环境”：先在设备列表左上方登录当前环境。
- 提示“当前环境登录已失效”：重新登录后再刷新。
- 提示“请先配置 ProductKey”：补齐设备的 `ProductKey`。
- 如果是 Video：确认当前选择的产品协议和视频模式一致，GB28181 产品不要拿去配 RTSP / RTMP，RTSP 产品也不要拿去配 RTMP。
- 其他失败：优先检查当前环境网关是否可访问、产品是否存在。

## 5. 还能不能用自定义 JSON

可以。

即使当前环境未登录或平台暂时不可达，仍然可以切换到“自定义 JSON”继续发送调试报文。
