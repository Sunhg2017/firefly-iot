# 前端操作提示文案收口使用说明

> 适用角色：系统运维、租户管理员、产品配置人员、联调开发人员
> 更新时间：2026-03-22

## 1. 这次改了什么

本次前端页面没有调整业务入口和操作流程，只调整了页面上的说明文案：

- 删除了“为什么这样改”“旧版怎么迁移”这一类页面说明。
- 保留并强化了“下一步点哪里”“提交后会发生什么”“哪些前置条件还没满足”这一类操作提示。

## 2. 你会看到哪些变化

### 2.1 数据分析导出

- 页面只提示导出结果会到任务中心下载。
- 不再在页面里解释导出链路如何改造。

### 2.2 租户 OpenAPI 文档

- 页面顶部只保留订阅范围、统一网关路径、AppKey 签名和文档快照时间。
- 不再展示页面设计层面的说明。

### 2.3 设备消息页

- 选择设备后，页面直接提示属性、事件、服务调用如何继续操作。
- 服务调用区只提示先选服务、填入参，不再解释列表过滤逻辑的设计背景。

### 2.4 角色、用户、租户授权

- 页面只提示怎么勾选角色、权限和租户空间功能。
- 不再解释为什么表单采用分步、为什么不再使用旧方式。

### 2.5 设备标签、协议解析、系统设置、OpenAPI 管理

- 页面提示改为搜索后绑定、确认识别字段、刷新后查看同步结果、按分组编辑配置等直接操作信息。

## 3. 操作建议

如果你在页面里看到提示语，可以直接按下面方式理解：

- 提示前置条件：
  说明当前还缺少设备、签名参数、服务或权限范围，需要先补齐。
- 提示操作结果：
  说明点击保存、提交、清空、导出后会发生什么。
- 提示访问范围：
  说明当前页面只显示你当前空间、租户或订阅范围内可操作的数据。

## 4. 模拟器说明

设备模拟器本轮已复查：

- 当前工作台、新建设备抽屉、设备列表和调试面板中的提示语仍以操作说明为主。
- 如需继续优化模拟器文案，建议只补充“如何连接、如何发送、如何查看结果”这一类提示，不再把设计背景放回界面里。
