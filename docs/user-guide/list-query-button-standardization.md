# 列表页查询按钮使用说明

## 适用范围

本说明适用于产品、设备、告警、字典、模板、OTA、视频等带搜索框的列表页。

## 使用方式

1. 在列表页顶部输入关键字，或先选择其他筛选条件。
2. 点击搜索框右侧的“查询”按钮，或继续使用回车触发查询。
3. 当需要刷新当前筛选结果时，无需刷新整个浏览器页面，直接再次点击“查询”即可。

## 说明

- 本次调整不会改变原有筛选条件含义。
- 已有的分页、状态筛选、下拉筛选仍按原方式生效。
- “查询”按钮的作用是为当前列表提供明确、可重复触发的刷新入口。

## 注意事项

- 若筛选结果为空，请先确认关键字和其他筛选条件是否正确。
- 如果点击“查询”后数据未更新，可检查是否存在接口报错或网络异常。
