# 设备数据页设备选择使用说明

## 现在怎么查设备数据

进入 `设备数据` 页面后，不需要再手工填写设备 ID，直接按设备名称搜索并选择设备即可。

## 各页签使用方式

### 最新数据

1. 点击设备选择框
2. 输入设备名称或别名关键字
3. 选中目标设备
4. 点击“查询最新值”

### 历史数据

1. 选择设备
2. 如有需要再补充属性名
3. 点击“查询”

### 聚合统计

1. 选择设备
2. 输入属性名
3. 选择聚合粒度
4. 点击“聚合查询”

### 设备事件

- 设备筛选现在也是下拉选择，不再要求输入设备 ID
- 可以只按级别筛选，也可以同时按设备和级别筛选
- 列表会直接显示设备名称

## 使用变化

- 页面不再要求记忆或输入数据库主键
- 事件列表默认展示设备名称，而不是设备 ID
- 搜索下拉会保留你已经选过的设备，切换关键词后不会丢失当前选择

## 常见问题

### 为什么我搜不到设备？

先确认：

1. 当前租户下已创建该设备
2. 设备名称或别名输入无误
3. 设备没有被禁用或删除

### 为什么事件里显示“设备信息缺失”？

说明这条历史事件对应的设备记录当前已经不存在，需要联系运维确认设备是否被删除。
