# 产品设备接入使用说明

## 1. 入口

进入产品列表，在目标产品上点击：

- 普通 IoT 产品：`设备接入`
- 摄像头产品：`视频接入`

## 2. 普通 IoT 产品怎么用

### 一机一密

1. 打开 `设备接入`
2. 按提示去设备管理创建设备
3. 获取设备自己的 `deviceSecret`
4. 按页面给出的 MQTT / HTTP / CoAP 参数接入

### 一型一密

1. 打开 `设备接入`
2. 在动态注册页填写设备名称
3. 执行动态注册
4. 使用返回的 `deviceSecret` 接入

## 3. 摄像头产品怎么用

1. 点击 `视频接入`
2. 页面会跳到 `设备管理` 的视频设备视图
3. 系统自动打开新增抽屉
4. 产品和协议会自动带入
5. 补充视频设备参数后保存

## 4. 注意事项

- 摄像头产品不使用 `ProductSecret`
- 摄像头产品不支持动态注册
- 视频设备现在统一归 `设备资产` 管理，不再有独立视频页
