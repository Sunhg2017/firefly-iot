# 设备模拟器开发启动说明
> 适用角色: 本地开发、联调、测试
> 更新时间: 2026-03-14

## 1. 为什么之前会弹出两个窗口

不是页面自己开了两个业务窗口，而是开发模式下 Electron 被启动了两次。

- 一次来自 `vite-plugin-electron` 自动启动
- 一次来自旧脚本里的手动 `electron .`

所以会看到两个窗口，其中开发态窗口通常还会自动打开 DevTools。

## 2. 现在应该怎么启动

```bash
cd firefly-simulator
npm run electron:dev
```

直接执行这一条命令即可，不需要再额外执行 `electron .`。

## 3. 现在的行为

- 正常情况下只会出现一个主窗口
- 如果重复启动，系统会优先聚焦已有窗口
- DevTools 自动打开仍然是开发态的正常行为

## 4. 如果还是看到两个窗口怎么办

优先检查：

1. 是否还有旧的 Electron 进程没有退出
2. 是否在别的终端或 IDE 任务里又启动了一次 `electron .`
3. 是否使用了未更新的历史启动脚本
