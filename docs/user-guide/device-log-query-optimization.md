# 设备日志使用说明
> 适用角色: 租户设备管理员、实施、运维
> 日期: 2026-04-06

## 1. 在哪里进入

租户空间 → 数据洞察 → 设备日志

## 2. 这个页面能做什么

页面用于查看设备运行过程中产生的日志，并支持：

- 按设备筛选
- 按产品筛选
- 按日志级别筛选
- 按日志内容关键字筛选
- 按时间范围筛选
- 查看完整日志详情
- 清理 30 天前的过期日志

## 3. 怎么查询

推荐按以下顺序操作：

1. 先选择设备，或者先选择产品缩小范围
2. 再选日志级别
3. 必要时输入日志内容关键字
4. 如果要定位时间段问题，再补时间范围
5. 点击“查询”

说明：

- 设备框支持搜索设备名称和别名
- 页面内部仍使用 `deviceId` 查询，但不要求手工输入

## 4. 列表怎么看

列表会直接展示：

- 设备：别名和设备名称
- 产品：产品名称和 `productKey`
- 级别
- 模块
- 内容摘要
- Trace ID
- IP
- 上报时间

如果日志关联的设备或产品已经不存在，页面会提示“设备信息缺失”或“产品信息缺失”。

## 5. 怎么看详情

1. 在日志列表中点击“详情”
2. 在弹窗中查看完整日志内容

详情里会展示：

- 设备名称 / 别名
- 产品名称与 `productKey`
- 日志级别、模块、Trace ID
- IP、上报时间、入库时间
- 完整日志内容

## 6. 怎么重置查询

点击“重置”后会清空以下条件：

- 已选设备
- 已选产品
- 日志级别
- 关键字
- 时间范围

页面会自动重新查询默认结果。

## 7. 清理过期日志时要注意什么

- “清理过期”只会清理当前租户 30 天前的日志
- 清理前建议先确认需要保留的时间范围
- 清理后如需复查，请先导出或记录关键日志内容
