# 产品物模型 GPS 定位仪模板使用说明
> 适用角色: 产品经理、设备接入实施、测试人员
> 日期: 2026-03-20

## 1. 这是什么

`GPS定位仪模板` 是产品物模型模板库里的一个预置模板，适合：

- 车载定位器
- 资产追踪器
- 人员定位终端
- 其它以经纬度上报和围栏告警为核心的定位设备

## 2. 模板会带出什么

应用模板后，系统会自动补出一组常见定位能力：

- 属性：纬度、经度、海拔、速度、航向角、卫星数、定位状态、电量、上报间隔
- 事件：SOS 告警、围栏告警、低电量告警
- 服务：实时定位、设置上报间隔

平台原有的固有项，例如 `ip`、`online`、`offline`、`heartbeat`，也会继续保留。

## 3. 怎么使用

1. 进入“产品与物模型”页面。
2. 找到目标产品，点击 `物模型`。
3. 在物模型抽屉里点击 `模板库`。
4. 找到 `GPS定位仪模板`。
5. 根据场景选择：
   - `追加模板`：在当前物模型后继续追加定位能力
   - `覆盖当前`：直接用 GPS 定位仪模板替换当前自定义内容

## 4. 什么时候适合用

推荐在这些场景直接使用：

- 设备主要上报当前位置、速度、围栏状态
- 需要快速打通定位设备接入
- 团队希望先用标准字段起步，再按业务补充细节

## 5. 注意事项

- 已发布产品只能追加模板，不能覆盖现有物模型。
- 如果你的设备还需要油量、ACC、里程、基站信息等字段，可以在应用模板后继续补充。
- 如果只是普通环境传感器，不建议直接使用 GPS 定位仪模板。
