# 租户管理使用说明

## 1. 适用角色

- 系统运维
- 平台管理员

## 2. Webhook 入口

在“租户管理”列表中，每个租户行都提供 `Webhook` 操作入口。

## 3. 套餐与配额能力已下线

- 当前版本不再提供租户套餐、配额查看或配额调整能力。
- “租户管理”列表中不再提供套餐筛选、套餐列、“调整套餐”或“调整配额”入口。
- 如仍看到相关入口，说明页面资源还未刷新到最新版本。

## 4. 如何维护租户 Webhook

1. 打开“租户管理”
2. 找到目标租户
3. 点击该行的 `Webhook`
4. 在抽屉中新增或编辑 Webhook
5. 填写地址、请求方法、Content-Type 和可选签名密钥
6. 保存后可直接使用“测试”校验连通性

## 5. 注意事项

- 这里只维护当前租户自己的 Webhook
- Webhook 不会出现在平台默认通知渠道页面
- 若租户没有配置 Webhook，通知发送不会自动回落到平台 Webhook

## 6. 租户编码复用说明

- 租户注销后，原租户会进入逻辑删除状态。
- 已注销租户的 `code` 现在可以再次用于创建新租户。
- 如果重新创建相同 `code` 时仍提示重复，请联系运维确认系统是否已完成最新数据库迁移。

## 7. 租户注销入口

- 在“租户管理”列表的“更多”菜单中，`注销租户` 现以停止态图标展示，便于和普通配置类操作区分。
- 当列表操作项较多时，可先通过图标快速定位危险操作，再进入二次确认。
