# 设备分组与设备主流程联动使用说明

## 适用角色

- 租户设备管理员
- 运维人员

## 可以做什么

现在设备分组已经接入设备主流程，可以直接在设备管理中完成这些操作：

- 新建设备时选择所属分组
- 编辑设备时调整所属分组
- 批量导入设备时统一加入分组
- 在设备列表按分组筛选设备
- 在设备分组页管理分组结构和成员

## 设备管理页怎么用

## 1. 新建设备时绑定分组

1. 打开“设备管理”
2. 点击“新建设备”
3. 选择所属产品
4. 填写设备名称、设备别名
5. 在“所属分组”中直接多选已有分组
6. 提交后，设备会立即加入这些分组

## 2. 编辑设备分组

1. 在设备列表找到目标设备
2. 点击“编辑”
3. 在“所属分组”中增删分组
4. 保存后立即生效

## 3. 按分组筛选设备

1. 打开“设备管理”
2. 在筛选区使用“所属分组”下拉框选择目标分组
3. 列表会只显示该分组下的设备

补充说明：

- 也可以直接点击设备表格中的分组标签，快速切换到该分组筛选结果

## 4. 批量导入设备时统一分组

1. 点击“批量导入”
2. 选择所属产品
3. 上传 Excel 或 CSV 文件
4. 在“统一分组”中选择一个或多个分组
5. 提交后，导入成功的设备会自动加入这些分组

## 设备分组页怎么用

## 1. 创建分组

1. 打开“设备分组”
2. 点击“新建分组”
3. 输入分组名称
4. 选择上级分组或保持为空作为顶级分组
5. 填写分组描述并保存

注意：

- 当前版本只支持静态分组

## 2. 把设备加入分组

1. 在左侧树中选择目标分组
2. 在“分组成员”区域搜索设备名称或别名
3. 选择设备后点击“加入设备”

## 3. 从分组移除设备

1. 选择目标分组
2. 在成员列表中找到目标设备
3. 点击“移除”

## 4. 删除分组

1. 选择目标分组
2. 点击“删除”
3. 确认后，当前分组及其子分组会一起删除

注意：

- 删除分组不会删除设备本身，只会删除分组结构和成员关系

## 使用建议

- 分组适合表达区域、产线、楼层、项目阶段、运维责任边界
- 如果一个设备同时属于多个业务维度，可以加入多个静态分组
- 设备标签更适合表达属性特征，设备分组更适合表达管理结构
