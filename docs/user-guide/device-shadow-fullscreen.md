# 设备影子卡片全屏查看使用说明
## 1. 适用角色

- 设备运维人员
- 产品调试人员
- 设备接入实施人员

## 2. 新增了什么

在“设备影子”页面中，下面四张卡片都新增了“全屏”按钮：

- Desired / 期望值
- Reported / 上报值
- Delta / 差异
- Metadata / 元数据

点击后可以放大当前卡片内容，便于查看长 JSON、做字段比对和排查。

## 3. 如何使用

1. 进入“设备影子”页面并先选择设备。
2. 查询到影子内容后，在任意卡片右上角点击“全屏”。
3. 在弹层中查看完整内容。
4. 关闭弹层后回到原页面继续操作。

## 4. Desired 卡片的特殊说明

Desired 既支持普通卡片内编辑，也支持全屏后继续编辑。

你可以在全屏态中直接执行：

- 编辑
- 保存
- 取消
- 清空

全屏中的内容和卡片中的内容始终是同一份草稿，不需要担心两边不同步。

## 5. 其他三张卡片的说明

Reported、Delta、Metadata 即使进入全屏，也仍然是只读查看模式，不支持人工修改。

## 6. 常见场景

### 6.1 比对 Desired 与 Reported

当影子 JSON 很长时，可以分别打开 Desired 和 Reported 的全屏窗口，逐段检查字段差异。

### 6.2 查看 Delta

当页面提示设备存在待同步差异时，可以打开 Delta 全屏查看具体是哪些字段还没有追平。

### 6.3 排查 Metadata

当需要确认字段更新时间或来源时，可以打开 Metadata 全屏查看完整元数据内容。
