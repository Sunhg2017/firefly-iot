# 前端页面说明文案调整说明

## 1. 调整目的

页面现在优先把空间留给筛选、编辑、发送、保存和结果反馈，不再在页面里重复展示大量背景说明。

## 2. 你会看到的变化

- 很多页面的页头不再显示“先做什么、再做什么”说明。
- 一些“说明 / 使用规则 / 操作提示”卡片被移除或压缩。
- 协议接入页顶部保留了相关入口和端点信息，但不再显示长段引导文案。
- 设备影子、设备消息等工作区会保留错误、状态和操作结果，但减少解释性文字。

## 3. 不变的部分

以下信息仍然保留：

- 必填校验和格式错误提示
- 删除、清空等危险操作确认
- 发送成功、发送失败、保存失败等结果反馈
- 当前对象的动态状态、统计数据和空状态

## 4. 使用建议

- 直接根据页面上的按钮、字段和标签完成操作。
- 遇到失败时优先看字段校验、结果提示和确认弹窗。
- 如果页面缺少完成当前操作所必须的信息，再反馈具体页面和操作步骤。
