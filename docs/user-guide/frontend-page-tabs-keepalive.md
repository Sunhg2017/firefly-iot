# 前端页面标签与保活使用说明
> 适用角色: 系统运维管理员、租户管理员、租户业务用户
> 日期: 2026-03-16

## 1. 功能说明

系统会在页面顶部展示“已打开页面”标签。你从一个业务页面切换到其他页面后，可以直接通过顶部标签返回，而且之前尚未提交的输入内容、筛选条件和局部编辑状态会继续保留。

页面顶部右侧的通知、异步任务和用户名会固定显示在同一行，用户名区域为轻量行内样式，不会再出现独立胶囊背景。

## 2. 如何使用

1. 进入一个业务页面并开始填写内容或调整筛选条件。
2. 通过左侧菜单继续打开其他业务页面。
3. 在页面顶部找到刚才打开过的标签。
4. 点击任意标签，直接切换到对应页面并继续之前的操作。

## 3. 关闭标签

- 非首页标签支持直接关闭。
- 关闭当前标签后，系统会自动切换到相邻标签。
- 如果当前没有其他已打开标签，系统会回到当前空间首页。

## 4. 注意事项

- 页面保活仅对当前登录会话内、尚未关闭的标签生效。
- 如果浏览器整页刷新，页面会重新初始化，未提交内容不会继续保留。
- 如果页面本身在进入时主动重置表单，以页面自身业务逻辑为准。
- 错误页和无权限页面不会加入顶部标签。
- 打开第二个及后续页面时，系统会直接激活新标签并展示对应页面，不会再自动跳回第一个已打开标签。
- 系统运维空间中的“定时任务”菜单会显示时钟图标；如仍未显示，请先刷新前端版本。
