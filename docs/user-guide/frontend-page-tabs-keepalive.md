# 前端页面页签与保活使用说明
> 适用角色: 系统运维管理员、租户管理员、租户业务用户
> 日期: 2026-03-16

## 1. 功能说明

系统现在会在页面顶部展示“已打开页面”页签。你从一个业务页面切换到其他页面后，可以直接通过顶部页签返回，而且之前尚未提交的输入内容、筛选条件和局部编辑状态会继续保留。

## 2. 如何使用

1. 进入一个业务页面并开始填写内容。
2. 通过左侧菜单切换到其他页面。
3. 在页面顶部找到刚才打开过的页签。
4. 点击该页签返回原页面，继续之前的操作。

## 3. 关闭页签

- 非首页页签支持直接关闭。
- 关闭当前页签后，系统会自动切换到相邻页签。
- 如果当前没有其他打开页签，系统会回到当前空间首页。

## 4. 注意事项

- 页面保活只对当前登录会话内、尚未关闭的页签生效。
- 如果浏览器整页刷新，页面会重新初始化，未提交内容不会继续保留。
- 若页面本身在进入时主动重置表单，以页面自身业务逻辑为准。
- 错误页和无权限页面不会加入顶部页签。
- 系统运维空间中的“定时任务”菜单会显示时钟图标；如果仍未显示，请先刷新前端静态资源版本。
