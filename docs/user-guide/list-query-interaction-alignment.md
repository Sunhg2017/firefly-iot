# 列表查询区使用说明

## 现在怎么查列表

所有已完成统一的列表页都按下面的方式使用：

1. 首次打开页面时，系统会自动查一次默认列表
2. 你可以先把关键词、状态、分类、协议等条件一次性调好
3. 调整完成后点击 `查询`
4. 如果想恢复默认条件，点击 `重置`

## 使用规则

- 改输入框内容时，列表不会立刻刷新
- 改下拉筛选时，列表也不会立刻刷新
- 只有点了 `查询`，当前条件才会真正生效
- 翻页时会继续沿用上一次点击 `查询` 的条件

## 查询区怎么识别

统一后的查询区有两个特征：

- 筛选项集中放在左侧
- `查询 / 重置` 等按钮固定放在右侧动作区

## 常见问题

### 为什么我改了下拉框，列表没立刻刷新？

这是当前统一规则。先把条件调好，再点 `查询`，可以避免连续改条件时重复请求。

### 为什么翻页后条件还在？

因为分页会沿用当前已生效筛选，避免查到下一页时条件突然丢失。

### 为什么有的页面还是旧样式？

如果页面还没切到这套统一查询区，先刷新静态资源；如果刷新后仍旧如此，说明该页面仍在旧版本，需要继续升级前端代码。
