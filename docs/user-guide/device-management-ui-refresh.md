# 设备管理与设备影子界面优化使用说明
> 适用角色: 租户管理员、运维、联调开发
> 日期: 2026-03-14

## 1. 设备管理页现在怎么看

设备管理页现在分为三层：

1. 顶部统计卡：快速看设备总数、在线、离线、禁用数量。
2. 中部筛选区：按设备名称、产品、设备状态、在线状态筛选，并直接看到当前筛选摘要。
3. 设备目录：在表格中直接查看设备名、所属产品、`productKey`、状态以及影子等操作入口。

## 2. 设备影子怎么用

### 2.1 独立影子页

- 不再需要手工输入设备 ID。
- 直接搜索设备名称或别名后选择设备，再点击“查询影子”即可。
- 查询后可看到：
  - 影子版本
  - 期望属性数量
  - 上报属性数量
  - 待同步差异数量

### 2.2 Desired 维护

- 点击“编辑”后可以直接修改 Desired JSON。
- 点击“保存”提交修改。
- 点击“清空”可清除当前 Desired。
- `Reported`、`Delta`、`Metadata` 区域为只读，用于观察设备实际状态和同步差异。

### 2.3 管理页里的影子抽屉

- 在设备管理页点击某台设备的“影子”即可打开抽屉。
- 抽屉头部会先展示设备名称、设备编码和所属产品，再展示影子概览、Desired、Reported、Delta 和 Metadata。
- 如果只想快速查看同步状态，优先看抽屉顶部概览即可。

## 3. 注意事项

- 影子版本只有在影子状态真正变化时才有意义，单纯刷新页面不会改变版本。
- 如果 `Delta` 不为空，表示设备当前上报状态还没有追平平台期望状态。
- Desired JSON 编辑时请保持合法 JSON 格式，否则系统会拒绝保存。
