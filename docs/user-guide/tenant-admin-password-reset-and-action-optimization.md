# 租户超管密码重置与列表操作优化使用说明
> 适用角色: 系统运维、平台管理员
> 日期: 2026-03-17

## 1. 功能说明

租户管理页面现在支持直接重置租户超级管理员密码，同时将低频操作折叠到“更多”菜单，列表会更清爽。

## 2. 如何重置租户超级管理员密码

1. 进入“租户管理”。
2. 找到目标租户所在行，点击“更多”。
3. 选择“重置超管密码”。
4. 在确认框中点击“重置密码”。
5. 系统会弹出新的随机密码，请在关闭前复制并通知对方登录后修改。

## 3. 如何理解新的操作布局

- 行内直接显示的是高频核心操作，如“用量、编辑、启用/暂停”。
- 套餐、配额、空间授权、Webhook、重置超管密码、注销等低频动作已统一收进“更多”菜单。

## 4. 注意事项

- 新密码只展示一次，关闭后不会再次显示。
- 如果误关了弹窗，需要重新执行一次“重置超管密码”。
- 系统运维租户不支持空间授权和 Webhook，因此其“更多”菜单项会比普通租户少。
