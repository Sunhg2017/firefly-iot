# 设备管理使用说明
> 适用角色: 租户管理员、运维人员
> 更新日期: 2026-03-14

## 1. 删除设备

在设备管理列表中点击“删除”后，系统会执行逻辑删除：

1. 当前设备会从设备列表中消失
2. 平台不再允许该设备继续按原记录参与管理
3. 产品设备数会同步更新

## 2. 删除后的表现

- 删除成功后，刷新列表不应再看到该设备
- 如果删除成功提示已出现，但列表仍有该设备，请刷新后再次确认
- 若刷新后仍存在，说明服务端删除链路异常，应联系运维按运维文档排查

## 3. 注意事项

- 删除操作不可直接恢复原设备记录
- 如果只是临时停用，请优先使用“禁用”而不是“删除”
- 已删除设备不会出现在常规设备列表中
