# 设备标签与设备主流程联动使用说明

## 适用角色

- 租户设备管理员
- 运维人员

## 设备管理页怎么用

### 新建设备

1. 打开“设备管理”
2. 点击“新建设备”
3. 选择所属产品
4. 填写设备名称、别名
5. 在“设备标签”中直接选择已有标签
6. 提交后，标签会直接绑定到该设备

### 编辑设备标签

1. 在设备列表中找到目标设备
2. 点击“编辑”
3. 在“设备标签”中增删标签
4. 保存后立即生效

### 批量导入设备

1. 点击“批量导入”
2. 选择产品
3. 上传 Excel 或 CSV 文件
4. 选择“统一设备标签”
5. 提交后，导入成功的设备会自动绑定这些标签

## 设备标签页怎么用

### 新建标签

1. 打开“设备标签”
2. 点击“新建标签”
3. 填写标签键、标签值、颜色、描述
4. 保存

### 给标签绑定设备

1. 在标签列表中点击“绑定设备”
2. 在抽屉中搜索设备名称或别名
3. 从下拉列表中选择设备
4. 点击“绑定”

说明：

- 不再需要手工输入设备 ID
- 已绑定设备会在下方列表中直接展示名称、别名、产品和绑定时间

### 解绑设备

1. 打开标签的“绑定设备”抽屉
2. 在已绑定设备列表中点击“解绑”

## 使用建议

- 标签键建议表达分类维度，例如 `region`、`env`、`line`
- 标签值建议表达具体取值，例如 `cn-east`、`prod`、`line-a`
- 同一维度尽量保持命名统一，避免出现语义重复的多个标签

## 注意事项

- 设备标签必须先在“设备标签”页面定义，设备页只能选择已有标签
- 标签绑定关系以系统维护为准，不需要再填写文本标签
- 删除标签后，相关设备上的该标签会一并移除
