# 设备影子版本持续增长处理说明
> 适用角色: 运维、联调开发、测试
> 日期: 2026-03-14

## 1. 问题表现

设备已经断开连接，但设备影子页面里的 `version` 还在不断增加。

## 2. 当前处理

- 模拟器断开后会自动停止自动上报，不再继续向服务端发送属性上报。
- 如果上报内容和当前影子 `reported` 完全一致，服务端不会再把影子版本继续加一。

## 3. 如何确认

1. 连接设备并开启自动上报。
2. 断开设备连接。
3. 刷新影子页面，确认 `version` 不再持续增长。
4. 如果版本仍增长，检查是否还有其他设备端或脚本在持续上报。

## 4. 注意事项

- 如果上报 payload 中包含时间戳、随机数等持续变化字段，即使设备在线，影子版本持续增长也是正常的。
