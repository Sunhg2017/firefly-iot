# 设备模拟器登录门禁、入口视觉优化与主页面缩放使用说明

## 1. 登录页现在怎么用

登录页调整为左右两区：

- 左侧显示当前环境概览，包括环境名称、当前网关、登录状态和环境数量。
- 右侧是登录卡，只保留环境切换、账号密码输入和环境管理入口。

主操作集中在右侧登录卡，进入路径更直接。

## 2. 现在怎么进入模拟器

启动模拟器后，如果当前环境未登录，不会直接进入主工作台，而是先进入登录页。

进入步骤：

1. 先确认当前环境。
2. 输入用户名和密码。
3. 点击“登录并进入主页面”，也可以直接按回车提交。
4. 登录成功后，系统自动进入主工作台。

## 3. 怎么切换环境

- 直接在右侧登录卡切换当前环境。
- 点击“环境管理”可以打开环境管理抽屉，进行更完整的环境维护。

环境管理抽屉支持：

- 新增环境
- 编辑环境
- 删除环境
- 设为当前环境

切换到其他环境后，登录卡会自动刷新该环境对应的用户名预填，并清空密码。切换到未登录环境后，需要重新登录才能进入主页面。

## 4. 主页面缩放现在是什么规则

- 主页面不再按响应式拆列。
- 窗口放大、缩小时，整套主框架会一起等比例缩放。
- 左侧设备区、中间主工作区、右侧工具和日志区会保持固定相对比例。

这意味着缩小窗口时，主页面会整体变小，但不会再出现只剩左右两侧、中间区域被挤没的情况。

## 5. 退出登录后会怎样

- 在主工作台退出登录后，会立即返回登录页。
- 返回登录页后，可以重新登录，也可以先切换环境。

## 6. 使用建议

- 该界面按 PC 场景设计，建议在桌面窗口中使用。
- 如果需要查看更多细节，优先放大窗口；系统会按比例整体放大主舞台。
