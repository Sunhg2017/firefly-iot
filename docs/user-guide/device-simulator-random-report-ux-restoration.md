# 设备模拟器随机模拟上报说明
> 适用角色: 联调开发、测试、实施
> 日期: 2026-03-14

## 1. 为什么你会感觉这个功能“没了”

随机模拟上报能力并没有删除，只是之前被藏到了 `Payload Template` 下拉里，而且默认不选模板，所以页面先显示的是自定义 JSON，看起来像只剩手工报文了。

现在已经恢复为显式入口。

## 2. 现在怎么用随机模拟上报

进入设备右侧的 `Data Report` 区域后：

1. 在“数据源”里选择 `随机模拟`
2. 选择一个“随机模板”
3. 查看“随机样例预览”
4. 如果想换一组示例，点 `换一组`
5. 点 `Send` 发送

## 3. 自动上报时会不会也随机

会。

只要当前模式是 `随机模拟`，无论你是手工点击 `Send`，还是打开 `Auto report`，每次发送都会重新生成一份新的随机 payload。

## 4. 什么时候该用自定义 JSON

以下场景更适合切到 `自定义 JSON`：

- 需要固定 payload 反复重放
- 需要验证某个极端值
- 需要构造后端异常场景

## 5. 如果某个类型没有随机模板怎么办

可以打开模拟器左下角的“模板管理”，新增对应的：

- 属性模板
- 事件模板
- OTA 模板

新增后回到 `Data Report`，就能在“随机模拟”里直接选到了。
