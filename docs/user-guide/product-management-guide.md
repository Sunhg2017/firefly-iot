# 产品管理页面使用说明

## 1. 适用角色

- 产品经理
- 设备接入实施人员
- 平台运维人员

## 2. 页面怎么用

产品管理页分成三层浏览逻辑：

1. 先看 `页面总览`
   - 快速了解当前筛选结果、已发布数量、开发中数量和当前页设备规模。
2. 再用 `筛选条件`
   - 通过关键词、分类、协议、状态快速缩小范围。
3. 最后按需要选择卡片视图或表格视图
   - 卡片视图适合快速浏览产品目录
   - 表格视图适合批量管理和精确查看字段

## 3. 卡片视图怎么看

单张产品卡片已经按“身份信息优先、辅助信息后置”的方式整理：

- 顶部角标：显示产品状态
- 标题区：显示产品名称、ProductKey、可选型号
- 标签区：显示分类、协议、认证方式
- 指标区：显示节点类型、数据格式、设备数量
- 说明区：显示产品说明和创建时间

建议阅读顺序：

1. 先看产品名称和 ProductKey，确认是不是目标产品。
2. 再看分类、协议、认证方式，判断它属于哪类接入模型。
3. 最后看节点类型、数据格式、设备数量，判断规模和技术形态。

## 4. 常用操作

### 4.1 新建产品

1. 点击页面右上角 `新建产品`
2. 在抽屉中填写产品名称、分类、协议、节点类型、数据格式、认证方式
3. 按需上传产品图片、填写型号和产品说明
4. 点击 `创建产品`

### 4.2 编辑产品

1. 在卡片或表格中点击 `编辑`
2. 修改产品基础信息
3. 点击 `保存修改`

### 4.3 管理物模型

1. 找到目标产品
2. 点击 `物模型`
3. 在物模型抽屉中维护属性、事件、服务

### 4.4 查看认证信息或动态注册

- 点击 `查看密钥` 或 `认证说明`，查看当前产品认证方式
- 已发布且使用一型一密的产品，可以点击 `动态注册`

### 4.5 进入协议解析

- 如果当前账号具备协议解析查看权限，可点击 `协议解析`
- 系统会自动带上当前产品上下文进入协议解析页

## 5. 使用建议

- 想快速找产品时，优先使用关键词搜索加状态筛选。
- 想按接入能力比对产品时，优先看卡片视图。
- 想进行批量管理或核对时间、字段时，切换到表格视图更高效。
- 产品说明建议写清楚用途、场景、安装方式，卡片视图会直接展示这部分内容。

## 6. 注意事项

- 已发布产品不能直接删除。
- 有设备关联的产品不允许删除。
- 当前页总览统计是当前页口径，不代表整库统计。
- 若首次进入页面后看不到想要的产品，先检查筛选条件是否仍然生效。
