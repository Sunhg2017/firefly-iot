# 视频监控页面使用说明

## 适用角色

- 视频接入实施人员
- 运维人员
- 联调测试人员

## 功能入口

- 从租户空间 `设备资产 -> 视频监控` 进入
- 或从摄像头产品页点击 `视频接入` 进入

## 页面怎么用

视频监控页分成三块：

1. 顶部状态区
   - 显示设备总数、在线数、离线数、GB28181 数量
2. 筛选区
   - 可按名称、设备编号、IP、接入方式、状态筛选
3. 设备列表区
   - 可播放、查询目录、查询设备信息、删除设备

## 从产品页进入时会发生什么

如果你是从摄像头产品页点击 `视频接入` 进入：

1. 页面顶部会显示当前产品上下文
2. 系统会自动打开“添加视频设备”抽屉
3. 接入方式会锁定为当前产品协议
4. 你只需要补充设备名称、地址、厂商型号等字段

注意：

- 当前联动只负责带入产品和协议信息
- 视频设备仍需在本页单独创建
- 当前不会自动建立视频设备与产品的持久化关联字段

## 如何添加视频设备

1. 点击页面右上角 `添加设备`
2. 填写设备名称
3. 选择或确认接入方式
4. 按协议补充参数
5. 点击 `创建设备`

### GB28181

需要重点填写：

- 设备名称
- GB 设备编号
- GB 域
- 传输协议
- IP 地址
- 端口

### RTSP / RTMP

需要重点填写：

- 设备名称
- IP 地址
- 端口
- 厂商
- 型号

## 如何播放和调试

1. 找到目标设备
2. 在线设备点击 `播放`
3. 在右侧抽屉中查看实时画面
4. 如需截图，点击 `截图`
5. 如需停止，点击 `停止`

### GB28181 专属动作

在线的 GB28181 设备还可以：

- 点击 `目录` 发起目录查询
- 点击 `信息` 发起设备信息查询
- 在播放抽屉里使用 `云台控制`

## 使用建议

- 设备名称尽量使用业务唯一名称，避免后续重复创建。
- 从产品页进入时，优先沿用页面推荐的协议，不要手工改成别的协议。
- 如果只是临时联调，结束后及时删除测试视频设备，避免误占设备数。

## 常见问题

### 为什么接入方式不能改？

因为你当前是从摄像头产品页联动进入，系统已经按产品协议锁定了接入方式。

### 为什么摄像头产品没有 ProductSecret？

因为摄像头产品统一走视频协议接入，不使用 ProductSecret 和动态注册。

### 为什么我创建了设备但不能播放？

先检查设备是否在线，再检查协议参数、IP、端口和流媒体服务状态是否正确。
