# 用户编辑表单回填使用说明
> 适用角色: 系统运维管理员、租户管理员
> 日期: 2026-03-17

## 1. 功能说明

用户管理页面的“编辑用户”抽屉现在会自动带出当前用户的已有信息，管理员无需重新手工输入历史内容。

## 2. 正确使用方式

1. 进入“用户管理”页面。
2. 在目标用户所在行点击“编辑”。
3. 等待抽屉加载完成后，查看系统自动回填的用户名、姓名、手机号、邮箱和角色。
4. 修改需要调整的字段后保存。

## 3. 使用结果

- 编辑已有用户时，页面会优先读取用户详情并完成回填。
- 如果详情加载失败，页面会退回使用列表中的已有数据回填，避免抽屉完全空白。
- 用户名在编辑态仍保持只读，避免误改登录账号。
- 切换到“角色分配”或“确认提交”步骤后，系统会继续保留已回填的角色，不需要重新选择。

## 4. 注意事项

- 如果角色未显示，通常是当前用户详情里没有角色信息或角色选项尚未加载完成。
- 如果保存后再次编辑仍看不到最新数据，请先确认本次保存是否成功，再检查是否为旧前端缓存。
