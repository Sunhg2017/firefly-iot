# 前端页面说明文案收口运维说明

## 变更内容

- 调整 `firefly-web` 多个页面的页头描述与说明卡片文案。
- 更新接入页公共引导组件，移除能力宣传标签。
- 在仓库级 `AGENTS.md` 中新增页面文案强制规则。

## 发布影响

- 仅影响前端展示文案，不涉及数据库、后端接口或菜单基础数据变更。
- 不需要额外环境变量、服务配置或 Flyway 脚本。

## 验证项

发布后建议重点检查以下页面：

- 登录页：左侧仅保留登录步骤和登录须知。
- 工作台：首页描述为操作顺序，不再出现风格说明。
- AppKey 管理、Open API 文档、OpenAPI 管理：页头描述为筛选、查看、调用步骤。
- 产品接入、租户管理、菜单配置、系统菜单权限管理：页头和提示改为操作限制或操作步骤。
- LoRaWAN、Modbus、SNMP、TCP/UDP、WebSocket：公共引导区不再展示能力宣传标签。

## 构建与回归

- 执行 `cd firefly-web && npm run build` 验证前端构建通过。
- 回归重点关注文案改动页是否仍存在换行溢出、卡片高度异常和按钮错位。

## 回滚方式

- 如需回滚，直接回退本次前端提交即可。
- 本次变更不涉及数据迁移，无需额外清理数据库。
